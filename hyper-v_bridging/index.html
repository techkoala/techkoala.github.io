Hyper-V 桥接模式

Hyper-V 没有 VM 的桥接模式可以选择，为了让虚拟机可以和宿主机同网段，需要搭建网桥。

创建网桥的步骤如下：

1

操作 -> 虚拟交换机管理器 -> 新建虚拟网络交换机 -> 选择外部类型 -> 创建虚拟交换机 -> 设置连接类型为内部网络 -> 应用

首先点击虚拟机右边的 Hyper-V 操作选项虚拟交换机管理器：
